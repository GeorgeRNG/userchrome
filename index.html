Firefox userChrome.css generator

Firefox userChrome customizations to remove things I don't like.
Most things here can be done with shortcuts or in the toolbar.
Go through the settings to select that you like, click Generate and follow the install instructions.
Firefox updates might break this but if it does break something I'll fix it.
This is on github, so if you want to contribute you can. Go here.

How to install

Click generate to download the file
Go to about:config and search for toolkit.legacyUserProfileCustomizations.stylesheets
If it's false, set it to true. This allows userChrome.css to work.
Click generate to download userChrome.css
Go to about:support, find Profile Folder and click Open Folder
If there isn't, create a folder called chrome, and go into it.
Put userChrome.css into it and restart firefox.
